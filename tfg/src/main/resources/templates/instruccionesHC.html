¿Cómo se juega?

    En este juego, tienes que mirar un grupo de aviones y decidir si dos o más aviones van a chocar.

    Cada avión tiene una dirección (hacia dónde está mirando) llamada heading.

    Si dos aviones están enfrentados entre si (sus cabezas se miran fijamente), existe un conflicto y pueden chocar

    No habrá conflicto si dos aviones no están enfrentados entre si

  • Ejemplo de conflicto: ➡️​ ⬅️
  • ejemplo de NO conflicto: ⬆️​ ⬆️ - ⬆️​ ⬇️​​
